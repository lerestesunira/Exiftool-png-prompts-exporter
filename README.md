# Exiftool-png-prompts-exporter
Extract all positive prompts of PNG's (from a folder and subfolder), to a text file (using Exiftool)
JUST REPLACE THE ACCESS PATH TO YOUR EXIFTOOL FOLDER


To run it, place the .py in your desired png folder, open the terminal from this folder and run the .py: 
python Extractpromptsfromfolder.py
